    # 项目命名

    项目名全部采用小写方式，以中线分隔。比如:my-project-name

    # 目录命名

    目录名参照上一条规则,有复数结构时,要采用复数命名法,比如说:scripts,styles,images,data-models

    # JavaScript文件命名

    所有js文件名，多个单词组成时,采用中划线链接方式,比如说:账号模型文件accoutn-model.js

    # css,scss文件命名

    多个单词组成时,采用中划线链接方式,比如说:retina-sprites.scss

    # HTML文件命名

    多个单词组成时,采用中划线链接方式,比如说:error-report.html
